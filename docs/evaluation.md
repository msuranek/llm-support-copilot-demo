# Evaluation Notes

## Co měřit

- čas do prvního draftu
- míra úprav agentem
- procento odpovědí poslaných bez zásadního přepsání
- počet eskalací na seniora nebo specializovaný tým

## Kdy výstup raději nepouštět dál

- když chybí dostatek kontextu
- když dotaz obsahuje refund, právní nebo bezpečnostní riziko
- když retrieval vrátí rozporné informace
- když model přidává neověřená tvrzení

## Praktické pravidlo

Nejlepší copilot neskrývá nejistotu. Umí ji rozpoznat a předat člověku.
